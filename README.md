# 🕷️ Miguel Shema Ngabonziza — Portfolio

My personal portfolio, themed after Spider-Man: Into the Spider-Verse (Miles Morales).
I'm a Computer Science + Cybersecurity student at **Grambling State University** and an
AI + Automation trainee, building AI/ML systems and full-stack products end to end.

> **Live:** [migztech.vercel.app](https://migztech.vercel.app)

---

## ✨ Highlights

- **Miles Morales "What's Up Danger" theme** — magenta/cyan duotone, comic halftone
  texture, chromatic-glitch headings, animated spider-web dividers, and comic-style
  card hovers (with a "THWIP!" accent).
- **Single-page layout** — Hero, About, Education (+ relevant coursework), Experience
  timeline, Projects, Technical Skills, Leadership & Involvement, Certifications,
  Resume (embedded PDF), and Contact.
- **Light / dark mode**, responsive design, and `prefers-reduced-motion` support.
- **One shared data layer** (`frontend_next/app/data/portfolio.ts`) drives the whole site.
